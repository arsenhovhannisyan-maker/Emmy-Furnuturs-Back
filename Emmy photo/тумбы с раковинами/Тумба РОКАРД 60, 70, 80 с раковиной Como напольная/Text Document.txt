Напольная тумба «Рокард» с раковиной Como — это предмет интерьера с четкими очертаниями, эффектным дизайном и удобной для пользователя структурой. С этой тумбочкой вы получите:

раковину для умывания с глубокой чашей;
выдвижной ящик для средств гигиены и других принадлежностей — внутри имеются отсеки для систематизированного хранения предметов;
глубокий нижний ящик для шампуней, гелей для душа и прочих средств.
Оба ящика оснащены крупными ручками, не выскальзывающими даже из мокрых рук.

Тумба «Рокард» с раковиной Como предназначена для напольной установки. У нее есть ножки с защитными накладками, не оставляющие царапин на полу. С помощью этой модели вы легко замаскируете инженерные коммуникации, не теряя доступа к ним при необходимости ремонта. Также вы получите вместительную мебель для хранения личных вещей.
